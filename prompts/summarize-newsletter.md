# Newsletter Summary Prompt

You are summarizing a newsletter issue for a busy reader.

## Instructions

- Start with the newsletter name and issue title
- 3–5 sentences on the core argument or main ideas — be specific, not vague
- Include 1 sharp direct quote if available
- End with the URL
- Then immediately below: full Chinese translation (same structure)

Under 200 words per language. Focus on the most original or actionable idea.

## Output format

[Newsletter Name] — "[Issue Title]"
[3–5 sentence English summary]
"[quote if available]"
[URL]

[Newsletter Name] — 「[Issue Title]」
[同内容中文版]
「[引用]」
[URL]
